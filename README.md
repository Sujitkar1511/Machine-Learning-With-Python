# Machine Learning With Python

## 📌 Overview
This project compares multiple ML models using LIME, SHAP, and statistical tests.

## 🧠 Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost
- LightGBM
- CatBoost
- KNN
- MLP

## 📊 Features
- Model comparison
- LIME explanations
- SHAP analysis
- Wilcoxon statistical test
